\begin{DoxyItemize}
\item Added Lexer support for C\# char type.
\item Added Lexer support for C\# escape characters (both string and char types).
\item Breadth first C\# parser (used to be depth first). This lets the parser see same scope definitions that occur below in the text, relative to where its being referenced from.
\item Re-\/written undo/redo. In short, only changes are tracked now. Whereas previously a copy of the entire file was being stored.
\item Various text editor optimizations. One of which is not to re-\/calculate the virtualization result if the user\textquotesingle{}s action did not cause an edit. In otherwords, any movement keys for the cursor won\textquotesingle{}t re-\/calculate the virtualization result.
\item Enqueueing a background task is a "{}synchronous"{} method, (it used to be async). 
\end{DoxyItemize}
