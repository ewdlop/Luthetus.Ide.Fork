\doxysubsection{Detailed Description}
This class is a hack and was made on (2024-\/03-\/09).~\newline
~\newline
. 

The reason for this class\+: This same hack existed prior, but as value type properties that were on the \doxylink{namespace_luthetus_1_1_text_editor_1_1_razor_lib_1_1_text_editors_1_1_models_aa2a2da032900659e3cf98e4b28fc476d}{Text\+Editor\+View\+Model}.~\newline
~\newline


The \doxylink{namespace_luthetus_1_1_text_editor_1_1_razor_lib_1_1_text_editors_1_1_models_aa2a2da032900659e3cf98e4b28fc476d}{Text\+Editor\+View\+Model} is an immutable type.~\newline
~\newline


Originally the value type properties were added to \doxylink{namespace_luthetus_1_1_text_editor_1_1_razor_lib_1_1_text_editors_1_1_models_aa2a2da032900659e3cf98e4b28fc476d}{Text\+Editor\+View\+Model} as a hack to permit the UI to mark the cursor as being \textquotesingle{}within view\textquotesingle{}, but not trigger a re-\/render.~\newline
~\newline


But, since \doxylink{namespace_luthetus_1_1_text_editor_1_1_razor_lib_1_1_text_editors_1_1_models_aa2a2da032900659e3cf98e4b28fc476d}{Text\+Editor\+View\+Model} is immutable, this hack was unreliable. At times, the value type that was changed, was from an out of date view model. And as result had no effect.~\newline
~\newline


By object-\/boxing the value types, the hack is more reliable.~\newline
~\newline
