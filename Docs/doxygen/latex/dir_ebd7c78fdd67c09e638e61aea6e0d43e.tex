{\itshape {\bfseries{\doxylink{namespace_luthetus_1_1_compiler_services}{Luthetus.\+Compiler\+Services}}}} is a Text Editor component library using the \href{https://dotnet.microsoft.com/en-us/apps/aspnet/web-apps/blazor}{\texttt{Blazor UI Framework}} for \href{https://dotnet.microsoft.com/}{\texttt{.NET}}\hypertarget{dir_ebd7c78fdd67c09e638e61aea6e0d43e_autotoc_md51}{}\doxysubsection{\texorpdfstring{Goal}{Goal}}\label{dir_ebd7c78fdd67c09e638e61aea6e0d43e_autotoc_md51}
The aim of \doxylink{namespace_luthetus_1_1_compiler_services}{Luthetus.\+Compiler\+Services} is to replicate the features of the \href{https://microsoft.github.io/monaco-editor/}{\texttt{Monaco Editor}} that powers \href{https://code.visualstudio.com/}{\texttt{Visual Studio Code}} using Blazor.
