\doxysubsection{Detailed Description}
Everytime one renders a unique \doxylink{class_luthetus_1_1_text_editor_1_1_razor_lib_1_1_text_editors_1_1_displays_1_1_text_editor_view_model_display}{Luthetus.\+Text\+Editor.\+Razor\+Lib.\+Text\+Editors.\+Displays.\+Text\+Editor\+View\+Model\+Display}, a unique identifier for the HTML elements is created. 

That unique identifier, and other data, is on this class.

This class is a "{}ui event broker"{} in order to avoid unnecessary invocations of \texorpdfstring{$<$}{<}see cref="{}\+IText\+Editor\+Service.\+Post"{}/.

For example, all Text\+Editor\+View\+Model have a Text\+Editor\+View\+Model.\+Tooltip\+View\+Model. A sort of \textquotesingle{}on mouse over\textquotesingle{} like event might set the current Text\+Editor\+View\+Model.\+Tooltip\+View\+Model. But, the event logic overhead is not handled by the Text\+Editor\+View\+Model, but instead this class, then the valid events are passed along to result in the Text\+Editor\+View\+Model.\+Tooltip\+View\+Model being set.
