\doxysubsection{Detailed Description}
This is a sort of the built in System.\+Globalization.\+Culture\+Info or System.\+IFormat\+Provider but specifically for the \doxylink{interface_luthetus_1_1_ide_1_1_razor_lib_1_1_terminals_1_1_models_1_1_i_terminal}{ITerminal}. 

There are a variety of ways that the terminal output is being shown, and as such there is no single \textquotesingle{}string\textquotesingle{} value associated with the terminal.

Instead, through the use this type, one caches their own \textquotesingle{}string\textquotesingle{} value. And, when the terminal has new output, and one tries to read their cached format output, it is first recalculated and cached once again before being returned.

i.\+e.\+: the string value associated with a \doxylink{interface_luthetus_1_1_ide_1_1_razor_lib_1_1_terminals_1_1_models_1_1_i_terminal}{ITerminal} is lazily calculated.
