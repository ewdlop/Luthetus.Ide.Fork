{\itshape {\bfseries{\doxylink{namespace_luthetus_1_1_text_editor}{Luthetus.\+Text\+Editor}}}} is a Text Editor component library using the \href{https://dotnet.microsoft.com/en-us/apps/aspnet/web-apps/blazor}{\texttt{Blazor UI Framework}} for \href{https://dotnet.microsoft.com/}{\texttt{.NET}}\hypertarget{dir_0cd17818026a7be52077794beeee002c_autotoc_md191}{}\doxysubsection{\texorpdfstring{Goal}{Goal}}\label{dir_0cd17818026a7be52077794beeee002c_autotoc_md191}
The aim of \doxylink{namespace_luthetus_1_1_text_editor}{Luthetus.\+Text\+Editor} is to replicate the features of the \href{https://microsoft.github.io/monaco-editor/}{\texttt{Monaco Editor}} that powers \href{https://code.visualstudio.com/}{\texttt{Visual Studio Code}} using Blazor.

One can make use of this Text Editor by referencing the Nuget Package or cloning the source code and referencing it directly.\hypertarget{dir_0cd17818026a7be52077794beeee002c_autotoc_md192}{}\doxysubsection{\texorpdfstring{Getting started}{Getting started}}\label{dir_0cd17818026a7be52077794beeee002c_autotoc_md192}
