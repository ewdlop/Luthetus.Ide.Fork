\begin{DoxyItemize}
\item A free and open source IDE
\item Runs on Linux, Windows, and Mac
\item Written with the .NET environment\+: C\#, \href{https://dotnet.microsoft.com/en-us/apps/aspnet/web-apps/blazor}{\texttt{Blazor UI Framework}}, and a \href{https://github.com/tryphotino/photino.Blazor}{\texttt{lightweight Photino webview}}.
\end{DoxyItemize}
