 Abstract System.\+Reflection.\+Metadata\+Load\+Context usage to a single method invocation.~\newline
~\newline
 

This method invocation uses reflection to make a readonly \textquotesingle{}cache\textquotesingle{} of the runtime types.~\newline
~\newline


The \textquotesingle{}cache\textquotesingle{} is more so just a representation of the runtime types as if the source code itself were passed to the \doxylink{class_luthetus_1_1_compiler_services_1_1_c_sharp_1_1_parser_case_1_1_c_sharp_parser}{Luthetus.\+Compiler\+Services.\+CSharp.\+Parser\+Case.\+CSharp\+Parser}~\newline
~\newline


The idea is to create the \textquotesingle{}cache\textquotesingle{} once per .NET version that a user requests to reference. Then re-\/use the \textquotesingle{}cache\textquotesingle{} until the application is closed.~\newline
~\newline


(re-\/creation of the cache would be done each time the application is opened.)~\newline
