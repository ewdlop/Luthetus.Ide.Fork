Remove\+Related\+Files\+From\+Parent is used for showing codebehinds such that a file on the filesystem can be displayed as having children in the Tree\+View.~\newline
~\newline
 In the case of a directory loading its children. After the directory loads all its children it will loop through the children invoking Remove\+Related\+Files\+From\+Parent on each of the children.~\newline
~\newline
 For example\+: if a directory has the children \{ \textquotesingle{}Component.\+razor\textquotesingle{}, \textquotesingle{}Component.\+razor.\+cs\textquotesingle{} \} then \textquotesingle{}Component.\+razor\textquotesingle{} will remove \textquotesingle{}Component.\+razor.\+cs\textquotesingle{} from the parent directories children and mark itself as expandable as it saw a related file in its parent.
