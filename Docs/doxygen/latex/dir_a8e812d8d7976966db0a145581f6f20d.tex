\begin{DoxyItemize}
\item Tracks the file system default namespace.
\item Interpolate file system namespace when creating a templated C\# file.
\item Create codebehinds for Blazor components with the \textquotesingle{}partial\textquotesingle{}, \textquotesingle{}Component\+Base\textquotesingle{}, and \textquotesingle{}using\textquotesingle{} automatically.
\item Display unique icons for unique folders. Example\+: "{}wwwroot"{} is a globe, and "{}\+Properties"{} is a settings gear.
\item Context Menu contains\+: Copy, Cut, Rename, Paste, Delete, "{}\+New Empty File"{}, "{}\+New Templated File"{}, and "{}\+New Directory"{}.
\item For .NET Solutions the Context Menu also gets\+: "{}\+Add New C\# Project"{}, and "{}\+Add Existing C\# Project"{}
\item For C\# Projects the Context Menu also gets\+: "{}\+Add Project Reference"{}, "{}\+Set As Startup Project"{}, and "{}\+Remove (no files are deleted)"{}
\item Nesting of \textquotesingle{}codebehind\textquotesingle{} like files. So nesting "{}\+My\+Component.\+razor.\+cs"{} behind "{}\+My\+Component.\+razor"{}.
\end{DoxyItemize}\hypertarget{dir_a8e812d8d7976966db0a145581f6f20d_autotoc_md130}{}\doxysubsubsection{\texorpdfstring{Input File Dialog\+:}{Input File Dialog\+:}}\label{dir_a8e812d8d7976966db0a145581f6f20d_autotoc_md130}
