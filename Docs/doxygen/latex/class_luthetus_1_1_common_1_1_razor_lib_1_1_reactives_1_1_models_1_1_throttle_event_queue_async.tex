When enqueueing an event, a batch\+Func is also provided.~\newline
~\newline
. 

The batch\+Func is to take the \textquotesingle{}to-\/be-\/queued\textquotesingle{} entry, and decide if it can merge with the last event in the queue, as to make a batched event.~\newline
~\newline


This batch\+Func is invoked over and over, until either a null \textquotesingle{}batch event\textquotesingle{} is returned. Or, there are no more entries in the queue to merge with.~\newline
~\newline


When a null \textquotesingle{}batch event\textquotesingle{} is returned, then the last item in the queue remains unchanged, and after it the \textquotesingle{}to-\/be-\/queued\textquotesingle{} is ultimately queued.~\newline
~\newline


Each invocation of the \textquotesingle{}batch\+Func\textquotesingle{} will replace the \textquotesingle{}to-\/be-\/queued\textquotesingle{} unless the \textquotesingle{}batch event\textquotesingle{} returned was null.~\newline
~\newline
