\doxysubsection{Detailed Description}
Anything parsed starts off as a statement, then goes on to invoke the expression loop if the syntax is deemed an expression. 

So, every test in \doxylink{class_luthetus_1_1_compiler_services_1_1_c_sharp_1_1_tests_1_1_smoke_tests_1_1_parsers_1_1_expression_tests}{Expression\+Tests} needs to be replicated in this file.

This file parses the inputs as statements first, and goes into the expression loop when it is deemed necessary.

This brings up the question... should parsing be "{}statement first"{} or "{}expression first"{}? As of making this comment the \doxylink{class_luthetus_1_1_compiler_services_1_1_c_sharp_1_1_parser_case_1_1_c_sharp_parser}{CSharp\+Parser} is "{}statement first"{}.
