Before a throttle event is enqueued, this method is invoked. This allows for batching of \textquotesingle{}this\textquotesingle{} throttle event instance, and the last entry of the queue.~\newline
~\newline
. 

This method is invoked from within a \textquotesingle{}lock(\+\_\+queue) \{ ... \}\textquotesingle{}.~\newline
~\newline


Example use\+: one reads an entry from the queue, its an event that write out a character. Perhaps this throttle event also writes out a character. Instead of invoking \textquotesingle{}text.\+Insert(character)\textquotesingle{} two times. One might want to invoke \textquotesingle{}text.\+Insert\+Range(both\+Characters)\textquotesingle{};~\newline
~\newline


Returning null means enqueue the recent\+Event without any batching, (leave the old event as it was within the queue)
