 If PositionIndex is within bounds of the SourceText.

char PeekCharacter (int offset)
 If (PositionIndex + offset) is within bounds of the SourceText.

Then the character within the string SourceText at index of (PositionIndex + offset) is returned and PositionIndex is unchanged.

Otherwise, ParserFacts.END_OF_FILE is returned and the value of PositionIndex is unchanged.

offset must be > -1.
char BacktrackCharacter ()
 If PositionIndex being decremented by 1 would result in PositionIndex being less than 0.

Then ParserFacts.END_OF_FILE will be returned and PositionIndex will be left unchanged.

Otherwise, PositionIndex will be decremented by one and the character within the string SourceText at index of PositionIndex is returned.
string ReadRange (int length)
 Iterates a counter from 0 until the counter is equal to length.

Each iteration ReadCharacter will be invoked.

If an iteration's invocation of ReadCharacter returned ParserFacts.END_OF_FILE then the method will short circuit and return regardless of whether it finished iterating to length or not.

Detailed Description

Provides common API that can be used when implementing an ITextEditorLexer for the TextEditorModel.

The marker for an out of bounds read is ParserFacts.END_OF_FILE.
