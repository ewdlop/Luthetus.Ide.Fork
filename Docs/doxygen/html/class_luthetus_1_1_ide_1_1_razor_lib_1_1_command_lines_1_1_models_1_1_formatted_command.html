string Luthetus.Ide.RazorLib.CommandLines.Models.FormattedCommand.Tag = string.Empty
getset

It is uncertain whether this tag property is a good idea. It allows one to easily re-use an Terminals.Models.IOutputParser.

.

By, the output parser being provided the Terminals.Models.TerminalCommand which caused the output.

But, determining the command that was ran would require parsing the HACK_ArgumentsString, because even though TargetFileName would be for example "git", one still doesn't know if it was "git add" or "git commit" etc...

Therefore this Tag can be used to identify more specifically what command was ran, allowing the Terminals.Models.IOutputParser to parse accordingly.
