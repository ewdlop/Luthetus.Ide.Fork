Detailed Description

I'm not sure how to go about property initialization of an object versus collection initialization. For now I'll rather hackily set the ExpressionNode only if a comma immediately follows it.

Then to know if an instance of this type represents collection initialization one can check IsCollectionInitialization to know whether the token was 'default' or not.
