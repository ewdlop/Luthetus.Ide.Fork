Introduction

Luthetus.TextEditor is a Text Editor component library using the Blazor UI Framework for .NET

Goal

The aim of Luthetus.TextEditor is to replicate the features of the Monaco Editor that powers Visual Studio Code using Blazor.

One can make use of this Text Editor by referencing the Nuget Package or cloning the source code and referencing it directly.
