enum Luthetus.TextEditor.RazorLib.Edits.Models.CursorDisplacementKind

This enum describes which direction the cursor's column index moves based on the amount of text that was inserted, deleted, or some other form of editing.

.

CursorDisplacementKind.Left is -1 * (amount of text that was inserted, or etc...)

CursorDisplacementKind.None is 0 * (amount of text that was inserted, or etc...)

CursorDisplacementKind.Right is 1 * (amount of text that was inserted, or etc...)

  • Given a 'delete' key edit. If one performs an 'undo' then the CursorDisplacementKind of the 'insert' edit will be CursorDisplacementKind.None. (in the previous example the 'insert' edit is made because it is the opposite action to a 'delete' edit. This therefore will satisfy the 'undo')

  • Given a 'backspace' key edit. If one performs an 'undo' then the CursorDisplacementKind of the 'insert' edit will be CursorDisplacementKind.Right.

  • Given an 'insert' edit. If one performs an 'undo' then the CursorDisplacementKind of the 'delete' edit will be CursorDisplacementKind.Left.
