◆ ClearFireAndForget()

void Luthetus.Ide.RazorLib.Terminals.Models.ITerminal.ClearFireAndForget ( )

This will invoke the ITerminalOutput.ClearOutput method, by using '_ = Task.Run(...)'.

This will execute EVEN IF a command in the queue is currently being executed.

The fire and forget is because the terminal ITerminalOutput wraps any state mutation in a 'lock(...) { }'. So, the fire and forget is to avoid freezing the UI.
