This version adds an extra function invocation for no reason. To use IEnvironmentProvider.AbsolutePathFactory(string, bool) may be a negligible optimization however.

Keep this method here, it provides more clear documentation on how to create an instance of IAbsolutePath. Having to invoke a method on the IEnvironmentProvider is a bit hard to find.
